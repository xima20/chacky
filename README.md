# Chacky
## サイト概要
中国茶愛好家が同士でお茶会を簡単に開催できるサイト

### サイトテーマ
Meat With Chacky

### テーマを選んだ理由
コロナの影響で中国茶愛好家同士の対面でのお茶会が開催できなくなっている。
手軽にリモートお茶会を開催、参加できるコミュニティサイトを作成したいと考えた。


### ターゲットユーザ
中国茶愛好家

### 主な利用シーン
お茶会の開催
お茶セミナーの開催
